# Memory Architecture

This project uses distributed memory files for continuity across human and AI collaboration.

- `project-state.md` — current sprint status
- `decision-registry.md` — accepted decisions
- `failure-registry.md` — known anti-patterns
- `horizon-log.md` — deferred ideas
- `concept-map.md` — key concept links
- `ritual-intentions.md` — purpose and commitment anchors
- `prompt-lineage.md` — prompt evolution notes
