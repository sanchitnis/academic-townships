# Onboarding

## Quick start
1. Read `memory/project-state.md`.
2. Read `LOOP.md`.
3. Read `docs/charter_evolution/POLYCENTRIC_CHARTER_v0.1.md`.
4. Review `docs/charter_evolution/critiques/README.md`.
5. Pick a role from `docs/governance/ROLES.md`.

## Contribution rule
All substantial charter updates should include:
- a critique or synthesis update,
- governance traceability,
- and any relevant memory/wiki updates.
