# Roles

## Human roles
- **Project Steward / Committee Chair**: release gates and final decisions.
- **Committee Members**: review critiques, vote on tensions, approve drafts.
- **External Experts**: submit domain depositions and challenge assumptions.
- **Community Contributors**: comment and propose improvements.

## AI roles
- **Critique Generalist**
- **Governance Adversary**
- **Industry Integration Examiner**
- **Inclusion & Access Examiner**
- **Synthesis Aggregator**
